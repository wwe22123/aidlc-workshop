# 테이블오더 서비스 - User Personas

---

## Persona 1: 고객 (Customer)

| 항목 | 내용 |
|------|------|
| **이름** | 김민수 (대표 고객) |
| **역할** | 매장 방문 고객 |
| **나이대** | 20-60대 (폭넓은 연령층) |
| **기술 수준** | 초급~중급 (스마트폰/태블릿 기본 사용 가능) |
| **목표** | 대기 없이 빠르고 편리하게 음식을 주문하고, 주문 상태를 확인하고 싶다 |
| **불만 사항** | 직원을 기다리는 시간, 주문 실수, 메뉴판이 보기 어려움 |
| **사용 환경** | 매장 테이블에 설치된 태블릿 (브라우저 기반) |
| **핵심 니즈** | 직관적인 메뉴 탐색, 간편한 주문, 실시간 주문 상태 확인 |

---

## Persona 2: 매장 관리자 (Admin)

| 항목 | 내용 |
|------|------|
| **이름** | 박서연 (매장 사장님) |
| **역할** | 매장 운영자/관리자 |
| **나이대** | 30-50대 |
| **기술 수준** | 중급 (PC/태블릿 업무 사용 경험) |
| **목표** | 주문을 실시간으로 파악하고, 매장 운영을 효율적으로 관리하고 싶다 |
| **불만 사항** | 주문 누락, 테이블 상태 파악 어려움, 메뉴 변경의 번거로움 |
| **사용 환경** | 카운터 PC 또는 태블릿 (브라우저 기반) |
| **핵심 니즈** | 실시간 주문 모니터링, 간편한 테이블/메뉴 관리, 매출 현황 파악 |
